## Overview

This project is a **multi-part Power BI analysis** of socio-economic and service-related data for **Maji Ndogo**.

**Part 1** focuses on exploratory data visualisation to provide an overview of:

* Population distribution (rural vs. urban)
* Types and usage of water sources
* Queue time patterns across days and hours
* Queue composition by gender and age groups
* Relationships between queue size and waiting time
* Crime distribution by type, gender, and location

The purpose of this phase is to identify patterns, trends, and potential data anomalies through visual analysis.

Additional parts will be added to expand the analysis and investigate findings in more depth.
